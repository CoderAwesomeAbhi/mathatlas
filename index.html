Algebraic Manipulation
Master the core language of competition math: setting up equations, substitution, elimination, and algebraic identity recognition.
AlgebraLevel 1: Foundations29 problems

📌 Recommended next module

After this, continue to: Ratios, Proportions & Percentages

Key Techniques

→Setting up equations from word problems
→Substitution and elimination in systems
→Recognizing algebraic identities: (a±b)², a²−b²
→Simplifying complex fractions
→Clever rearrangement to reveal structure

Worked Examples

System by elimination
2x+3y=16 and x−y=1. Multiply second by 3: 3x−3y=3. Add: 5x=19 → not clean. Instead multiply second by 2: 2x−2y=2. Subtract from first: 5y=14 → y=14/5. Re-examine — always check if substitution is cleaner.
Difference of squares
Compute 103×97. Write as (100+3)(100−3) = 100²−9 = 10000−9 = 9991. This is almost always faster than direct multiplication.
Hidden quadratic
x + 1/x = 3. Then x² + 1/x² = (x+1/x)² − 2 = 9−2 = 7. And x³+1/x³ = (x+1/x)³ − 3(x+1/x) = 27−9 = 18. Build higher powers from lower ones.

Ratios, Proportions & Percentages
Ratio chains, proportional reasoning, mixture problems, and percent change. These appear in almost every early-AMC problem.
AlgebraLevel 1: Foundations30 problems

📌 Recommended next module

After this, continue to: Exponents & Polynomials

Key Techniques

→Unifying ratio chains via LCM: A:B=2:3 and B:C=4:5 → A:C=8:15
→Mixture problems: weighted average
→Percent increase/decrease cascades
→Part-whole relationships
→Cross-multiplication and unit rates

Worked Examples

Ratio chain unification
A:B = 2:3 and B:C = 4:5. Multiply A:B by 4 and B:C by 3 to get B=12 in both: A:B:C = 8:12:15. Now A:C = 8:15 directly.
Mixture problem
Mix 30% solution and 70% solution to get 50% solution. Let x = amount of 30% and y = amount of 70%. Then 0.3x + 0.7y = 0.5(x+y) → 0.3x+0.7y = 0.5x+0.5y → 0.2y = 0.2x → x=y. Equal amounts.

Exponents & Polynomials
Laws of exponents, polynomial arithmetic, factoring strategies, and roots. The building blocks of all advanced algebra.
AlgebraLevel 1: Foundations30 problems

📌 Recommended next module

After this, continue to: Introductory Geometry

Key Techniques

→Exponent laws: aᵐ·aⁿ=aᵐ⁺ⁿ, (aᵐ)ⁿ=aᵐⁿ, a⁰=1
→Difference of squares: a²−b²=(a+b)(a−b)
→Perfect square: (a±b)²=a²±2ab+b²
→Sum/difference of cubes: a³±b³=(a±b)(a²∓ab+b²)
→Factoring by grouping

Worked Examples

Factor: x⁴−16
x⁴−16 = (x²)²−4² = (x²+4)(x²−4) = (x²+4)(x+2)(x−2). Always factor difference of squares twice if possible.
Simplify 2¹⁰⁰/(2⁹⁸+2⁹⁸)
Denominator = 2·2⁹⁸ = 2⁹⁹. So expression = 2¹⁰⁰/2⁹⁹ = 2.

📘
Introductory Inequalities
Ordering, absolute value, and first exposure to inequality reasoning in competition problems.
AlgebraLevel 1: Foundations30 problems

📌 Recommended next module

After this, continue to: Quadratics & Vieta's Formulas

Key Techniques

→Triangle inequality: |a+b| ≤ |a|+|b|
→Absolute value cases: split on sign
→Non-negativity: x²≥0 for all real x
→Integer bounds from inequalities
→Estimation and bounding arguments

Worked Examples

Integer solutions via bounds
Find integers with 2x+1 > 7 and x < 6. From first: x > 3, so x ≥ 4. Combined with x < 6: x ∈ {4,5}. Two solutions.
